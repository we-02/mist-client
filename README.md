
# Mist Client (ALPHA)

Free and opensource Crystal PvP ghost client.


## Features

- AutoCrystal
- Anchor Macro
- Safe Anchor
- ESP
- PopChams
- and much more and soon to come..


## Installation

Download the `.jar` file from releases and add to your 1.21.11 Fabric instance

    
## Tested Servers

- Any Grim PvP server should theoretically be bypassed (tested on test.ccbluex.net)
- CatPvP
- USPvP
